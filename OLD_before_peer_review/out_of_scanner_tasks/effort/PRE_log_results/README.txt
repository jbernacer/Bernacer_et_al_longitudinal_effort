This folder contains all log files pertaining the effort-discounting task before the physical activity plan.

After completing the task (which was split into two sessions), the log file was produced. It contains 4 columns:

1) Money of the discounted option
2) EFfort of the discounted option
3) Position of the discounted option (-150=left; 150=right)
4) Choice (97=left; 98=right)
