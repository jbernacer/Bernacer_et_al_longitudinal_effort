This folder contains log files and FSL regressor files to run the fMRI analyses of this project.
The task pairs presented to each volunteer are also here (task_pairs.xlsx).

task_pairs.xlsx: Excel file with one sheet per participant. Each sheet contains 8 columns:
1) Task pair number
2) Probability of option A (% of winning the reward)
3) Effort level (minutes running) of option A.
4) Number of times option A was chosen.
5) Probability of option B
6) Effort level of option B
7) Number of times option B was chosen.
8) Missing responses.
